### Step 4 :
End the Program.

## Program:
```
#Program to find the inverse of a matrix.
#Developed by: SHYAM S
#RegisterNumber: 23012478
import numpy as np
A=np.array([[6,2,3],[3,1,1],[10,3,4]])
result=np.linalg.inv(A)
print(result)
```
